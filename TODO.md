## How to Run

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Start the backend:
```bash
python main.py
```

3. Open dashboard:
- Navigate to `frontend_dashboard/index.html` in your browser
- Or use a local server: `python -m http.server 8000` in the frontend_dashboard folder
